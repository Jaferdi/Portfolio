# Minimal Lit Portfolio

A polished full-stack developer portfolio built with pnpm, Vite, Lit Web Components, and Tailwind CSS.

## Install

```bash
pnpm install
```

## Run Locally

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Customize

Edit the profile, projects, skills, and experience in `src/data/content.js`.
